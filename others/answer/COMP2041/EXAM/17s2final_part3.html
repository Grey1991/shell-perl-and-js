COMP[29]041 Practice Exam - Part 3 of 3

October 2017

Time allowed for this part: 3 hours

Time allowed for entire exam: 3 hours on final exam

Total number of questions this part: 4

Total number of questions entire exam: 21

Total number of marks this part: 44

Total number of marks entire exam: 100

Answer all questions

Answering any of the questions correctly in this part counts towards the exam hurdle requirement of answering 2 questions correctly to pass COMP[29]041.

You will not be able to enter your answers for part 3 until after the first 35 minutes. After the first 35 minutes you can run programs such as editors and xterms.

Answer each question in a 'SEPARATE' file. Each question specifies the name of the file to use.

Make sure you use 'EXACTLY' this file name.

Submit these files using the give command as described in each question. You may submit your answers as many times as you like. The last submission 'ONLY' will be marked.

If you have only a partial solution and the exam is about to end, submit it anyway.

During the first 35 minutes the only programs you can run in the exam environment are the applications to enter answers to parts 1 and 2 of the exam and view part 3 of the exam clock (xclock), calculator (xcalc) and firefox to access cheatsheets and other documentation.

After the first 35 minutes the exam environment allows you to run a limited set of commands so you can test and debug your answers.

The exam environment logs all your keystrokes. Biometric keystroke timing data will be collected to confirm your identity.

The exam environment does not allow access to files in your CSE account or any other CSE account. It does not allow access to the web or other network access

All questions in this part must be answered in either Shell, Perl or Python. You may not use other programming languages, e.g. you may not use C or Java.

Some language documentation is available via the right mouse button menu. Manual entries are also available. No other documents or files will be available to you.
